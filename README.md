# Forti Demo

Демонстрационный проект по управлению/просмотру пациентов и их лекарственных рецептов.

Представь, что ты врач, который хочет найти:

* пациента А,
* список выданных ему рецептов,
* найти среди них опредлённые рецепты,
* и т. д.

## Системные требования

* **Ubuntu:**: 25.04
* **Ruby:** 3.4.1
* **Rails:** 7.0.10
* **PostgreSQL:** 17.7

## Быстрый старт

### 1. Клонируем проект
```bash
git clone https://github.com/ooobodryak/forti_demo.git
cd forti_demo
```
### 2. Устанавливаем зависимости
#### 2.1. Ruby
Например, используем `rbenv`.
```bash
rbenv install 3.4.1
rbenv local 3.4.1
```
#### 2.1. Ruby On Rails
```bash
gem install rails -v 7.0.10
```

#### 2.2. PostgreeSQL
```bash
sudo apt update
sudo apt install postgresql-17 postgresql-contrib libpq-dev

# ВАЖНО!
sudo -u postgres createuser -s имя_вашего_пользователя
```

#### 2.3. Gems
```bash
bundle install
```

### 3. Подготавливаем приложение к запуску
```bash
# Создаём базу данных.
rails db:create

# Выполняем миграции.
rails db:migrate

# Наполняем БД тестовыми данными.
rails db:seed
```

### 4. Запускаем приложение
#### 4.1. Запускаем сервер
```bash
rails s
```

#### 4.2. Открываем в бразуере
```bash
http://127.0.0.1:3000
```

Мы попадём на стартовую страницу со списком всех Пациентов, на которой можно:
* перейти на страничку Рецептов,
* фильтровать записи, например, по именам, количеству рецептов,
* удалять записи,
* и другое.

Страницы с фильтрацией сделаны полностью динамическими на `Stimulus` (**пагинация** + **фильтрация** + **удаление**).

## Знакомство с кодом
В проекте установлен гем `yard`, который позволяет генерировать документацию и смотреть её прямо в бразуере.

### 1. Смотрим документацию по всему проекту:
#### 1.1. Генерируем
```bash
yardoc
```

#### 1.2. Открываем в браузере

`doc/top-level-namespace.html`

До `doc` нужно указать путь этого файла в вашей системе, чтобы бразуер сообразил откуда его взять и открыть.

### 2. Делаем выборочную документацию

Для такого варианта достаточно передать `yard` относительный путь до просматриваемого вами Руби файла/-ов, например:
```bash
yardoc app/controllers/sample 
```

и открыть в браузере файлик `doc/Sample.html`.
